### Installing

Make sure that you have proper react native setup

```bash
cd app
```
```bash
npm i
```
```bash
cd server
```
```bash
npm i
```

### Run project

## For front end
```bash
cd client
```
```bash
npm start
```

## For server

Open new terminal
```bash
cd server && cp .env.example .env
```

```bash
npm i
```
